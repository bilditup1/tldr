# tldr
Minimalist [tldr](https://tldr.sh/) command line client, written in plain C99.

# FEATURES
* It builds.
* And then works.
* Under 400 lines.
* No dependencies.

# QUICK START
Clone this repository and enter it:

```
git clone https://github.com/unInstance/tldr
cd tldr
```

Compile and install tldr:

```
make
sudo make install
```

Run
```
tldr --update
```
